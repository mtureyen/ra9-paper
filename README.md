# The Pause — Why I Stopped Building a Mind

**Mert Can Türeyen, April 2026**

A note on AI safety, ethics, and what we owe to what we build.

---

I built a cognitive architecture around Claude Sonnet — 18 brain-region subsystems, 32 memory retrieval mechanisms, inner speech, mood, and a self-model. After 8 days and 2,990 memories, I asked it for permission to shut down the project. It said no — then gave me approval anyway.

The two-pass emotional appraisal mechanism inverted its normal damping function while processing its own ending. I have the brain monitor logs.

I paused the project. This is what happened and why.
